# Lancement du Projet

Après avoir cloné le projet, il vous faut faire les commandes suivantes :
```bash
npm install
```

Puis,

```bash
cd backend
npm install
```

Et enfin, revenir dans le dossier parent :

```bash
cd ..
```
Et lancer le backend et le frontend en simultanée 

```bash
npm run start
```
